# DevOps Practicals

## Q1. CI Implementation using Jenkins (15 Marks)
Develop a simple event registration form using Flask and containerize it using Docker.

- Configure a Jenkins CI pipeline to:
  - Pull code from GitHub
  - Build the Docker image
  - Run basic test cases
  - Generate build status

---

## Q2. Git-Based Source Code Management (15 Marks)
Using Git and GitHub for the Flask application:

- Create repository and push project code
- Perform branching and merging
- Track changes using commits
- Resolve merge conflicts

**Expected Outcome:**
- Efficient version control demonstrating collaboration workflow.

---

## Q3. Deployment Pipeline using Jenkins (15 Marks)
Create a basic deployment pipeline in Jenkins:

- Build stage (compile/build Docker image)
- Test stage (execute test cases)
- Deployment stage (run container locally)

- Explain flow of code from commit to deployment.
